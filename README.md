JournalApp to konsolowy e-dziennik dla szkół.

Wymagania funkcjonalne:
    - zarządzanie kontami uczniów i nauczycieli (administrator) dodawanie/usuwanie/edycja ról
    - zarządzanie przedmiotami (administrator) dodawanie/usuwanie/edycja
    - dodawanie ocen oraz wpisów (nauczyciel)
    - wyświetlanie ocen, obliczanie średniej (uczeń/nauczyciel)
    - zapisywanie danych w formacie JSON (symulowanie bazy danych)

Wymagania pozafunkcjonalne:
    - walidacja ocen (1-6),
    - przechowywanie danych w formacie JSON

Odbiorcy:
    - uczniowie
    - nauczyciele
    - dyrekcja (administrator)

Korzyści:
    - automatyzacja obliczeń
    - trwałośc danych
